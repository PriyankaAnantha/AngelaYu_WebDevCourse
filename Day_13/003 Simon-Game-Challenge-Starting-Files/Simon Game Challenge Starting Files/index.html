Welcome to the Simon Game

Game Description

This is an exciting game where you need to follow the sequence of colors and repeat them. Each level gets progressively harder. Can you keep up?

How to Play

  1. Press any key to start the game.
  2. Watch the sequence of colors.
  3. Repeat the sequence by clicking the buttons in the correct order.
  4. If you make a mistake, the game will restart.
  5. Try to get the highest score possible!
